# A Social Media Website

# DevSocial

DevSocial is a full-stack social media web application where users can create posts, share images or videos, and browse feed.

The goal of this project was to build a modern, scalable social platform while strengthening full-stack development skills.

---

## Key Features

- User authentication (register, login, logout)
- Create posts with text, images, or videos
- Responsive UI for mobile and desktop
- Infinite scrolling feed (loads content smoothly)
- Secure media uploads using cloud storage
- Clean and modern user interface

---

## Tech Stack

### Frontend

- React (with TypeScript)
- Tailwind CSS
- Zustand (state management)

### Backend

- Node.js and Express
- MongoDB
- JWT Authentication
- Cloudinary (for image and video storage)

---

## What I Focused On

- Building a clean and responsive UI
- Handling real-world features like file uploads and authentication
- Writing scalable and organized backend APIs
- Implementing smooth user experience with infinite scrolling

---

## Project Highlights

- Designed a social media feed similar to modern platforms
- Implemented efficient data loading using cursor-based pagination
- Integrated image and video uploads with cloud storage
- Built reusable components and clean state management
